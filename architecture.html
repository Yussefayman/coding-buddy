AI Agent / Architecture
Click any component to inspect · Toggle flow direction above
── User Layer
entrypoint
CLI
Click-based entrypoint. Single-prompt or interactive REPL. Owns Agent lifecycle via async context manager.
presentation
TUI
Rich console renderer. Streams text deltas, renders syntax-highlighted tool panels with metadata.
── Orchestration Layer
agent
Agent
Drives the agentic loop. Emits AgentEvents upstream. Delegates to Session for all stateful work.
session
Session
Groups LLMClient, ContextManager, and ToolRegistry under a session UUID with turn tracking.
── Services Layer
client
LLMClient
Async OpenAI-compatible client. Assembles streaming tool call deltas. Retries on rate limits.
state
ContextManager
Ordered message history. Prepends system prompt. Tracks token counts per message via tiktoken.
tools
ToolRegistry
Registers tools, validates Pydantic schemas, dispatches invocations, exports OpenAI schemas.
── External & Base Layer
external
LLM API
Any OpenAI-compatible endpoint. Receives messages + tool schemas, returns SSE completions.
base
Config / Loader
Merges system → project TOML configs. Injects AGENT.md as developer instructions.
base
Tools (Builtin)
Tool ABC with Pydantic schemas. ReadFileTool + extensible registry. Async execute().
